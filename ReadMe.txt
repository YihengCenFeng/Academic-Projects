g++ -std=c++11 MazeSolver.cpp main.cpp

-------------------------------------------------------------

Input Files:
	"inputA.txt"
	"inputB.txt"
	"inputC.txt"
	"inputD.txt"

Note: input file can be changed through main.cpp

-------------------------------------------------------------

Maze characters:
	_ = hallway
	> = path
	* = wall
	@ = backtrack
	$ = exist

Note: the starting position will be in the upper-left position
